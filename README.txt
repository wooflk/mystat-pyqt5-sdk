# MyStat SDK

## Описание

MyStat SDK - современное десктопное приложение для работы с системой MyStat, предоставляющее удобный интерфейс для просмотра оценок, посещаемости, расписания и домашних заданий.

## Возможности

- Аутентификация через MyStat API
- Просмотр оценок и статистики
- Отслеживание посещаемости
- Просмотр расписания занятий
- Управление домашними заданиями с загрузкой файлов
- Таблица лидеров
- Современный фиолетово-оранжевый дизайн
- Адаптивный интерфейс

## Технические требования

- Python 3.7+
- PyQt5
- SQLAlchemy
- Requests

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python run.py`

## Архитектура

### Главное приложение (main.py)
- LoginWindow - окно входа в систему
- MainWindow - главное окно приложения
- DashboardWidget - дашборд с обзором
- SidebarButton - кнопки навигации
- CardWidget - карточки контента

### Логика API и БД (core.py)
- Конфигурация базы данных
- Конфигурация API
- Утилиты для работы с HTTP
- API функции для получения данных

### Модели данных (models.py)
- User - пользователи
- Mark - оценки
- Attendance - посещаемость
- Homework - домашние задания
- Schedule - расписание

### Интерфейс API (interface/mystat_interface.py)
- MystatInterface - основной класс для работы с API
- Методы аутентификации
- Методы получения данных
- Обработка ошибок

### Виджеты (widgets/)
- GradesWidget - отображение оценок
- AttendanceWidget - статистика посещаемости
- ScheduleWidget - расписание занятий
- HomeworkWidget - домашние задания
- CalendarWidget - календарь событий

## Использование

1. Запустите приложение командой `python run.py`
2. Введите логин и пароль от MyStat
3. Используйте боковую панель для навигации
4. Просматривайте данные в соответствующих разделах

## API

Приложение использует MyStat API для получения данных:
- Аутентификация: `/v1/mystat/auth/login`
- Оценки: `/v1/mystat/{campus}/statistic/marks`
- Посещаемость: `/v1/mystat/{campus}/statistic/attendance`
- Расписание: `/v1/mystat/{campus}/schedule/get-month`
- Домашние задания: `/v1/mystat/{campus}/homework/list`
- Загрузка файлов: `/v1/mystat/{campus}/user/file-token`
- Отправка заданий: `/v1/mystat/{campus}/homework/create`
- Таблица лидеров: `/v1/mystat/{campus}/progress/leader-table`

## База данных

Приложение использует SQLite для локального хранения данных:
- Кэширование пользовательских данных
- Сохранение сессий
- Локальная статистика
